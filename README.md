### 디렉터리 구조도

```bash
📦src
 ┣ 📂api
 ┣ 📂constants // 공통적으로 사용되는 상수들 정의폴더
 ┣ 📂hooks // 커스텀 훅 폴더
 ┣ 📂styles // css 파일 폴더
 ┣ 📂types // 타입 재정의 하는 폴더

 ┣ 📂pages // 페이지 컴포터는 폴더
```
